GUIDE
-------------------------------------------------------------------------------------
- web tool to play with various types of chaos attractors
- set various parameters
- choose a color palette
- show equations for the attractor
- pause/resume/reset simulation
-------------------------------------------------------------------------------------
How to use (preferred):
 - visit link https://simon-smida.github.io/vin-attractool/
 - hope it is still being hosted (if not, my mail is: xsmida03@vutbr.cz)
 - enjoy!


How to use (naive):
 - download all the 3 files (index.html, style.css, script.js) into the same folder
 - open index.html (in your favorite web browser)
 - enjoy!
